# **0.声明** #
此项目内所有取自原神的素材版权归米哈游所有。该项目仅用于个人学习研究。  

# **1.简介** #
独立完成的个人mmorpg学习项目,包含客户端服务端。  
UI风格参考原神,使用Lua开发,支持热更新。  
客户端： Unity + XLua    
协议： protobuf     
服务端: C# + sqlserver

# **3.功能模块** #  
1.用户模块：用户注册、用户登录、角色创建、角色选择  
2.基础模块：角色控制、移动同步、地图传送、小地图  
3.基础系统：背包系统、装备系统、商店系统、任务系统、Npc系统、对话系统   
4.社交系统：好友系统、邮件系统、组队系统
